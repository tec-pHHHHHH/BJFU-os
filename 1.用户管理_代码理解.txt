来看第一步的用户管理的代码：
现在这一堆.c文件里面全是一个个函数，函数声明全是在对应的.h头文件中，然后主函数.c文件会包含所有的头文件，这样就相当于把所有.c文件中定义的函数包含在主函数中了。

用户管理user.c

首先是一个叫做user_hash_password的函数，传入两个参数，一个是用户输入进来的password，一个是一块地址，用来存放计算好的哈希结果。
这个函数没时间理解了，反正就是将用户输入的明文密码，通过一些列的操作，变成64位哈希字符串。然后存放到数据库用户表中。


好的我们来看user_find这个函数，这个函数用来更具用户名查找对应的用户，两个参数，一个是当前系统状态的结构体s，里面有用户链表的头节点s->user_list，；另一个是要查找的用户名。

先定义一个User类型的变量u，用来存放系统中第一个用户，也是用户链表的首元素，然后进入一个循环，调用strcmp函数来和要查找的用户名匹配。直到遍历完整个用户链表为止。找到了，就返回u这个结构体。

===user_register函数===
这个函数用来登记新用户，传入三个参数：系统状态结构体，用户名，密码。

错误检测。

用户名中只能由数字和字母以及下划线组成，一个for循环判断是否符合条件。

调用userfind函数，不能有重复的用户名。

接下来定义了一个int类型的变量count初值为0。定义一个用户结构体u指向系统用户链表首元素。一个while循环，遍历用户链表，每层循环count++，循环后count的值就是系统中用户的值。如果当前用户数量大于规定最大数量，报错。

新定义一个用户结构体new_user，并调用malloc函数给这个结构体画一块大小为用户结构体大小的空间。

接下来调用strncpy函数把用户输入的用户名写入结构体new_user中，最后一位是'\0'

调用user_hash_password函数把传入的密码转成64位哈希值存储。

（接下来了解下User这个结构体：
typedef struct User {
    char  username[MAX_NAME_LEN];
    char  password_hash[MAX_PASSWORD_LEN];  /* 简单哈希存储 */
    int   is_locked;         /* 账户是否被锁定(3次错误登录) */
    int   login_attempts;    /* 当前连续错误次数 */
    struct User *next;       /* 链表指针 */
} User;

包含用户名，密码的哈希，账户锁定判断位，当前错误次数记录位，还有链表指针用来存入系统用户链表中）

继续，把新用户的锁定位置为0，错误次数为9=0，next指针为空。

然后三行代码存入现有系统用户链表。头插法
1.新用户的下一个元素是原链表首元素
2.现在链表的手元素是新用户
3.系统的脏位 置为1。告诉系统有修改，需要保存。

最后调用user_save_to_file(S)函数，并正常退出。


===用户登录函数user_login===

依旧传入三个参数。包含系统状态的结构体s，传经来的用户名以及密码.

先一个判断，如果当前登录用户不是空值，直接退出。

接下来定义一个用户结构体u，调用user_find函数，传入s和username来查找已经登记的用户，返回值给u。u就包含这个用户的信息了。

判断是否锁住。

接下来调用之前的哈希函数把用户尝试登录时传入的密码转换成哈希值，再调用srtcmp函数，与储存的哈希值比较，如果错了的话尝试次数就++，三次错误，那么就把锁定位 置为1，用户锁上了。

如果密码对了的话，就重置登录尝试值，并且调用strcmp函数
把用户名传入当前登录用户数组中。



===用户登出函数user_logout===
只需要传入一个参数，对就是那个s。

一个判断，如果当前登录用户数组为空，就提示用户。

如果不为空，就把当前登录用户数组置空，然后提示用户。

===把所有用户数据写到一个文件里 user_save_to_file===

首先调用fopen函数打开要存入的文件，返回一个文件描述符fp。

定义count = 0，定义用户结构体类型的变量u，指向系统用户链表的首元素。一个while循环，count记录当前系统总有多少个用户。

调用fwrite函数，把count写入文件。

重新让u指回系统用户链表首元素。一个while循环，每层循环中调用fwrite函数，分别把这个用户的用户名，密码哈希，是否锁住，登录尝试次数存入文件，count++。

关闭文件描述符。

===把所有用户数据从文件中读取 user_load_from_file===

首先调用fopen函数打开要访问的文件，返回一个文件描述符fp。

只传入当前存有当前系统状态的结构体s。

定义一个count用来从文件中读取第一个字节，也就是用户数量。

一个for循环，循环count次，每层循环都定义一个User类型的用户结构体u，并调用malloc函数给它分配内存。

调用fread函数，分别从文件中读取用户名，密码哈希，是否锁住，登录尝试次数，并存入结构体u。

然后每层循环也要用头插法法把结构体u存入系统用户链表user_list

关闭描述符，返回count。
